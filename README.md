# 🛡️ VNC Shield AI

An AI-powered cybersecurity dashboard for detecting and preventing **VNC-based data exfiltration attacks** through behavioral analysis and intelligent monitoring.

---

## 🚀 Problem Statement

Modern organizations rely on remote access tools like **Virtual Network Computing (VNC)** for system management. However, attackers can exploit these sessions to silently capture sensitive data directly from screens.

This creates serious challenges:

* ❌ No file transfer → traditional security fails
* ❌ Signature-based detection is ineffective
* ❌ Logs can be manipulated after attacks
* ❌ No real-time behavioral monitoring

---

## 💡 Solution

**VNC Shield AI** introduces a smart monitoring system that:

* Tracks active VNC sessions in real-time
* Assigns threat scores based on behavior
* Detects anomalies using AI-based logic (concept)
* Stores logs securely using blockchain (concept)
* Enables admin-level control to block suspicious activity

---

## ✨ Key Features

* 📊 Cybersecurity Dashboard with real-time metrics
* 🔍 Live Session Monitoring Interface
* ⚠️ Threat Detection & Risk Scoring
* 🔗 Blockchain Logs (tamper-proof simulation)
* 🧠 AI-based anomaly detection concept
* ⚙️ Admin Panel with IP blocking & controls

---

## 🛠️ Tech Stack

* **Frontend:** React + Vite + Tailwind CSS
* **Backend:** FastAPI
* **Charts:** Recharts
* **Architecture:** AI + Cybersecurity + Blockchain concepts
