# 🛡️ VNC Shield AI

An AI-powered cybersecurity dashboard for detecting and preventing **VNC-based data exfiltration attacks** using behavioral analysis and intelligent monitoring.

---

## 🚀 Problem Statement

Modern organizations rely on remote access tools like **Virtual Network Computing (VNC)** for system management. However, attackers can exploit these sessions to silently capture sensitive data directly from screens.

This creates serious challenges:

* ❌ No file transfer → traditional security fails
* ❌ Signature-based detection is ineffective
* ❌ Logs can be manipulated after attacks
* ❌ No real-time behavioral monitoring

---

## 💡 Solution

**VNC Shield AI** introduces a smart monitoring system that:

* Tracks active VNC sessions in real-time
* Assigns dynamic threat scores based on behavior
* Detects anomalies using AI-based logic (concept)
* Stores logs securely using blockchain (concept)
* Enables admin-level control to block suspicious activity

---

## ⚙️ System Architecture

```text
User Activity → Session Monitoring → Feature Extraction
        ↓
   Threat Analysis Engine (AI Concept)
        ↓
  Dashboard Visualization → Admin Control
        ↓
 Blockchain Logging (Tamper-proof Concept)
```

---

## 🔍 How It Works

1. 📡 **Session Monitoring**
   Captures simulated VNC session data (IP, duration, activity)

2. 🧠 **Threat Scoring Engine**
   Applies rule-based logic to assign a threat score (0–100)

3. ⚠️ **Anomaly Detection (Concept)**
   Flags unusual patterns such as:

   * High session duration
   * Suspicious activity spikes
   * Unknown IP behavior

4. 📊 **Dashboard Visualization**
   Displays sessions, alerts, and analytics in real-time UI

5. 🔗 **Blockchain Logging (Concept)**
   Logs events in tamper-proof format for future audit
